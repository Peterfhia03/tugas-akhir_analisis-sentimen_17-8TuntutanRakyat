# 🎓 Tugas Akhir - ANALISIS SENTIMEN PUBLIK TERKAIT 17+8 TUNTUTAN RAKYAT MENGGUNAKAN MODEL HIBRIDA INDOBERT-BILSTM PADA TEKS AUTENTIK MEDIA SOSIAL X

## 📌 Deskripsi Project

Project ini merupakan aplikasi analisis sentimen berbasis **Transformer-Sequential** menggunakan model **IndoBERT + BiLSTM** untuk mengklasifikasikan opini masyarakat terhadap topik 17+8 Tuntutan Rakyat

Aplikasi terdiri dari:

* **Frontend** → antarmuka pengguna
* **Backend** → API & pemrosesan model untuk klasifikasi sentimen

---
## 🧱 Struktur Project

```
├── backend/
├── frontend/
├── README.md
```

---

## ⚙️ Teknologi yang Digunakan

* Frontend: React / Next.js
* Backend: Node.js / Express
* Machine Learning: PyTorch / IndoBERT / BiLSTM

---

## 🚀 Cara Menjalankan Project

### 1. Clone Repository

```bash
git clone https://github.com/USERNAME/NAMA-REPO.git
cd NAMA-REPO
```

---

### 2. Setup Backend

```bash
cd backend
npm install
npm run start
```

---

### 3. Setup Frontend

```bash
cd frontend
npm install
npm run dev
```
